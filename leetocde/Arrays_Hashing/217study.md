## 해시 테이블(Hash Table)이란?

데이터를 빠르게 저장하고, 빠르게 찾을 수 있도록 설계된 자료구조.

파이썬의 set과 dict가 내부적으로 사용하는 구조도 바로 이 해시 테이블임

C++의 해시 테이블

C++에서는 대표적으로 두 가지 STL 컨테이너가 해시 테이블 기반으로 동작한다:

컨테이너 특징
std::unordered_map (키 → 값) 딕셔너리 형태
std::unordered_set 중복 없는 집합
이 두 컨테이너는 내부적으로 해시 함수를 사용하여 데이터를 저장하고 검색함으로써 평균적으로 O(1)의 시간 복잡도로 빠른 접근을 제공함.

### 해시 함수(Hash Function)

해시 함수는 임의의 크기의 데이터를 고정된 크기의 해시 값으로 변환하는 함수.
이 해시 값은 데이터의 "지문"과 같아서, 데이터를 빠르게 찾을 수 있도록 도와줌.
